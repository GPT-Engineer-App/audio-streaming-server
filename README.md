# audio-streaming-server

Make a simple and highly easy to use and host audio streaming server that will host a webpage via public ip. The webpage should contain a html5 stream that contains the microphone audio from the server with options to start recording and stop recording with a file list

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/audio-streaming-server.git
cd audio-streaming-server
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
